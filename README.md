# Anomaly Detection in Excel Financial Data

## Description
This project analyzes multi-year financial data using Microsoft Excel and Power Query.
Statistical techniques such as Z-Score and IQR are applied to detect anomalies in expense
and revenue transactions.

## Tools Used
- Microsoft Excel
- Power Query

## Key Steps
- Data cleaning and preprocessing
- Z-Score based anomaly detection
- IQR based outlier detection
- Summary metrics and data validation rules
